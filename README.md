# Scripts
## ¿Qué son los scripts?
![](http://1.bp.blogspot.com/-yYdaJzT-mBw/UY0Y-EqVLLI/AAAAAAAACTY/Uoid6vKS0_w/s1600/Poundexclam.svg.png)  
Un scripts es un archivo con varias lineas de código necesarias que indican al entorno que lo ejecute qué hacer y cuándo hacerlo. Estos scripts se utilizan para realizar tareas repetitivas o programadas.
